## ðŸ§ª Running Tests

To run the scraper locally:

- Python 3.8 or later
- Install dependencies:
    ```bash
    pip install pytest
    ```
- Run:
    ```bash
    pytest -v
    ```
- If tests pass, include a screenshot in your PR.

- **Note: Some tests require a US VPN because realforeclose.com is geo-blocked.**
        **If not in the US, these tests will be skipped automatically.**
